# Models

Pre-trained edge detection models used as baselines for comparison against the Wide View Filter.

## DexiNed

Dense Extreme Inception Network for edge detection. A deep CNN trained on BIPED that produces crisp edge maps without post-processing.

- **Source:** https://github.com/xavysp/DexiNed
- **Paper:** X. Poma et al., "Dense Extreme Inception Network for Edge Detection," IEEE WACV, 2020.

## TEED

Tiny and Efficient Edge Detector. A lightweight model designed for real-time edge detection with a small parameter count.

- **Source:** https://github.com/xavysp/TEED
- **Paper:** X. Soria et al., "Tiny and Efficient Model for the Edge Detection Generalization," ICCV Workshop, 2023.

## pidinet

Pixel Difference Network for edge detection. Uses pixel difference convolutions to capture edge features efficiently.

- **Source:** https://github.com/zhuoinoulu/pidinet
- **Paper:** Z. Su et al., "Pixel Difference Networks for Efficient Edge Detection," ICCV, 2021.

## EDTER

Edge Detection with Transformer. A vision-transformer-based approach that captures global context for edge detection.

- **Source:** https://github.com/MengyangPu/EDTER
- **Paper:** M. Pu et al., "EDTER: Edge Detection with Transformer," CVPR, 2022.

## EdgeNAT

Edge detection using Neighborhood Attention Transformer. Applies neighborhood attention mechanisms to edge detection.

- **Source:** https://github.com/SH-Annn/EdgeNAT

## NBED

Neural Based Edge Detector.

- **Source:** https://github.com/LiuTingWed/NBED

## DiffusionEdge

Edge detection using diffusion models. Applies denoising diffusion probabilistic models to generate edge maps.

- **Source:** https://github.com/GuHuangAI/DiffusionEdge
- **Paper:** G. Ye et al., "DiffusionEdge: Diffusion Probabilistic Model for Crisp Edge Detection," AAAI, 2024.
